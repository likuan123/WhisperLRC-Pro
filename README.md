# WhisperLRC Pro

> Offline English Audio → Bilingual LRC Generator

## Features

- Offline speech recognition
- Faster-Whisper
- Argos Translate
- English → Chinese
- Export LRC
- Export SRT
- Export TXT
- Batch processing
- Drag & Drop
- GPU acceleration
- Windows GUI

```bash

python main.py
```
